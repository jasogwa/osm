# Open street map
Map GeoJson from open street map api to leaflets  

## Installation

```bash
cd server/
npm install
```

```bash
cd client/
yarn 
```

### 3. Run the following on **server** 
```
npm run dev
```

### 4. Run the following on **client** 
```
yarn start
```

### 5. Start app
```
localhost
```
